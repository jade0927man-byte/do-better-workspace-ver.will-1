# 매출 성장 분석 프레임워크

> 기반: Variance Analysis (Horngren, 2014), MoM Growth Rate, Time Series Decomposition

## 1. 차이 분석 (Variance Analysis)

**출처**: Horngren, C. et al. (2014). Cost Accounting: A Managerial Emphasis, 15th Edition. Pearson
**배경**: 관리회계의 표준 기법. 예산 vs 실적, 전기 vs 당기 비교에서 차이의 원인을 구조적으로 분해

**핵심**: "매출이 올랐다/내렸다"가 아니라 **왜** 올랐는지를 수량/단가/믹스로 분해

### 3요인 분해

```
총 차이 = 당기 매출 - 전기 매출

분해:
1. 수량 효과 = (당기 수량 - 전기 수량) x 전기 평균 단가
   → 더 많이 팔았는가?

2. 단가 효과 = (당기 평균 단가 - 전기 평균 단가) x 당기 수량
   → 더 비싸게 팔았는가?

3. 믹스 효과 = 카테고리 구성 변화에 의한 차이
   → 고가 상품 비중이 올라갔는가?
```

### 해석 가이드

| 패턴 | 의미 | 시사점 |
|------|------|--------|
| 수량+ 단가+ | 양적, 질적 성장 모두 | 건강한 성장 |
| 수량+ 단가- | 가격 할인으로 물량 확대 | 마진 확인 필요 |
| 수량- 단가+ | 프리미엄화 또는 고객 감소 | 고객 수 모니터링 |
| 수량- 단가- | 전반적 부진 | 원인 분석 시급 |

## 2. 전월 대비 성장률 (MoM Growth Rate)

**출처**: Wall Street Prep, Corporate Finance Institute - 재무 분석 표준 지표
**활용**: SaaS, 리테일, 스타트업 성장 측정의 기본 지표

### 핵심 공식

```
MoM 성장률 = (당월 - 전월) / 전월 x 100

예시:
- 1월 매출: 6,700만원
- 2월 매출: 9,900만원
- MoM = (9,900 - 6,700) / 6,700 x 100 = +47.8%
```

### 성장률 맥락 해석

| 성장률 | 리테일 기준 해석 |
|--------|----------------|
| +30% 이상 | 비정상적 고성장 (계절성/이벤트 확인) |
| +10~30% | 양호한 성장 |
| +0~10% | 안정적 유지 |
| -0~10% | 경미한 감소 (계절성 가능) |
| -10% 이상 | 주의 필요 (원인 분석) |

> 계절성이 강한 관광 상권은 MoM보다 YoY(전년 동월 대비)가 더 정확하나, 1년 미만 데이터에서는 MoM이 유일한 비교 지표

## 3. 시계열 분해 (Time Series Decomposition)

**출처**: Cleveland, R.B. et al. (1990). "STL: A Seasonal-Trend Decomposition". Journal of Official Statistics
**간이 적용**: 학술적 STL 모델 대신 주차별 집계로 동일한 인사이트 도출

### 시계열 3요소

```
매출 = 추세(Trend) + 계절성(Seasonality) + 잔차(Residual)

실무 간이 적용:
- 추세: 주차별 매출 그래프의 방향 (상승/하락/횡보)
- 계절성: 요일별 패턴 (주말 피크, 평일 저조)
- 잔차: 설명 안 되는 변동 (이벤트, 날씨 등)
```

### 주차별 분석 프레임

```
주차 정의:
- 1주차: 2/1~2/7
- 2주차: 2/8~2/14
- 3주차: 2/15~2/21
- 4주차: 2/22~2/28

비교 포인트:
- 주차별 매출 증감 추이
- 특이 주차 식별 (설 연휴, 공휴일)
- 요일별 편차 (주말 vs 평일 비율)
```

## 4. 카테고리 기여도 분석

**출처**: Pareto, V. (1896). 80/20 법칙 응용

### 카테고리별 성장 기여도

```
카테고리 기여도 = (카테고리 당월 매출 - 카테고리 전월 매출) / 전체 매출 증가분 x 100

예시:
- 전체 증가분: +3,200만원
- 커피포 증가분: +2,400만원 → 기여도 75%
- 스낵 증가분: +500만원 → 기여도 15.6%
- MD상품 증가분: +300만원 → 기여도 9.4%
→ 성장의 75%가 커피포에서 발생
```

### 카테고리 포트폴리오 매트릭스

**출처**: BCG Growth-Share Matrix (Henderson, 1970) 응용

```
                    MoM 성장률
                 높음              낮음
         ┌────────────────┬────────────────┐
    높   │      Star      │   Cash Cow     │
 매 음   │ (성장+고매출)   │ (안정+고매출)   │
 출      ├────────────────┼────────────────┤
 비 낮   │ Question Mark  │     Dog        │
 중 음   │ (성장+저매출)   │ (정체+저매출)   │
         └────────────────┴────────────────┘
```

## 5. 베스트셀러 분석

### 순위 변동 추적

```
                1월 순위    2월 순위    변동
오리지날 10T       1           1        -    (부동)
바나나우유맛       3           2        +1   (상승)
생일파티바닐라     2           3        -1   (하락)
시나몬약과맛       신규         5        NEW  (신규 진입)
```

### 집중도 분석

```
상위 N개 상품 집중도 = 상위 N개 매출 / 전체 매출 x 100

해석:
- TOP 3가 50% 이상: 히트 상품 의존 → 품절 리스크 관리
- TOP 3가 30% 미만: 분산된 포트폴리오 → 마케팅 집중 어려움
- TOP 10이 80% 이상: 파레토 패턴 → 상위 상품 재고 우선
```
