### AI 인사이트 제안 패널

각 View 하단에 AI가 데이터를 분석하여 "생각해볼 지점"을 최대 3개 제시한다.

| 유형 | 탐지 조건 | 예시 |
|------|-----------|------|
| 이상 탐지 | 전월 대비 급변 (±15% 이상) | "[채널]의 매출이 전월 대비 +23%인데, 마진율은 -2.1%p입니다" |
| 추세 경고 | 3개월 연속 같은 방향 | "[매장]이 3개월 연속 목표 미달입니다 (87% → 82% → 78%)" |
| 기회 발견 | 비중 vs 성과 불균형 | "[채널] 마진율이 전 채널 최고(41%)인데 매출 비중은 12%입니다" |

**설계 규칙**: 결론이 아닌 제안 톤. 모든 인사이트에 참조 데이터 명시. View당 3개 이하.
